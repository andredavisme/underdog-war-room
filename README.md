# Underdog War Room

Internal competitive intelligence and sales activation site for a new brand entering a mature industry.

🔗 **[View the site](https://andredavisme.github.io/underdog-war-room/)**

## What's Inside

- **Threat Intel** — How established competitors will respond and what they won't do
- **Sales Playbook** — 6-step prioritized response sequence for contested accounts
- **Battlecard** — Side-by-side Us vs. Them comparison
- **Objection Handler** — Accordion-style responses to the 5 most common pushbacks
- **Team Mindset** — Speed, proof, and focus principles for the underdog advantage

## Stack

Static single-page HTML — no build tools, no dependencies beyond Google Fonts CDN.
Hosted via GitHub Pages.

## Usage

Open `index.html` directly in a browser or deploy to any static host.
